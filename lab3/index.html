Lab 3 - File Structure and File Transfer

The main point of this lab is to learn how to properly create, function, and transfer files in our Art101 repository. This text represents task #3 in our Lab intructions.

Main Lab Content

Challenge

The main challenge of this lab was understanding the navigation of VSCode and our own files. We were challenged to write code that linked our labs to our homepage, then we had to learn how to link our own screenshots into a lab and caption them.

Problems

This lab was particularly simple for the both of us, because much of the work had already been done for us for us or in previous labs. However, one area that was a challenge was learning how to correctly display an image so that it did not layer itself with another image or seperate too far. Then, it was a challenge to figure out how to make proper captions underneath them.

Reflection

This particular lab was quite fun because I was able to use some of the CSS HTML skills I had used in previous labs. I was particularly interested with making sure my pictures were aligned correctly despite their different size. I put more effort into this assignment than I normally would because I had fun tinkering with CSS.

Results

I was able to successfully navigate and structure my files in VSCode. Then, I was able to create and further customize a seperate lab 3 site that included new challenges such as images, lists, and captions.
